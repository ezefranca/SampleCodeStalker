SampleCodeStalker is a small application that fetches information about Apple's sample code from the web and presents it nicely. It started as a small weekend proejct and is written completely in Swift.

## Features

- [x] Nicely present sample code projects
- [x] Check for new and updated sample code
- [x] Download sample code with one click
- [ ] Track changes to downloaded sample code

## Requirements

- Mac OS X 10.11+
- Xcode 7.2+
